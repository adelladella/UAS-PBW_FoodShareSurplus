🍞 FoodShare

📘 Integrasi Developer & API Route

Rest API Modern berbasis Laravel 13 untuk mempermudah integrasi data real-time dengan aplikasi mobile relawan.

GET /api/available-food Menampilkan seluruh data surplus layak konsumsi

Mengambil database list makanan premium berstatus available untuk ditampilkan pada landing page.

Response JSON (200 OK)
{
  "status": "success",
  "count": 3,
  "data": [
    {
      "id": 1,
      "donor_id": 2,
      "name": "Nasi Box Ayam Goreng",
      "category": "makanan_berat",
      "quantity": 30,
      "status": "Available",
      "expired_hours_left": 4
    }
  ]
}
POST /api/claim-food Klaim instan makanan surplus untuk lembaga

Mengajukan transaksi peminjaman/klaim stok surplus oleh lembaga sosial resmi.

Request Payload JSON (Header: Content-Type: application/json)
{
  "food_id": 1,
  "lembaga_id": 4,
  "quantity_claimed": 15,
  "method": "delivery",
  "notes": "Mohon dikirimkan segera sebelum jam makan siang panti asuhan."
}
Response JSON (201 Created)
{
  "status": "success",
  "message": "Klaim makanan berhasil diajukan untuk divalidasi admin.",
  "claim_id": 182
}
PUT /api/update-status/{id} Update status tracking & kurir relawan

Digunakan oleh aplikasi kurir relawan untuk memperbarui posisi & status distribusi.

Request Payload JSON
{
  "status": "in_delivery",
  "courier_name": "Budi Santoso",
  "courier_phone": "081299887766"
}
Response JSON (200 OK)
{
  "status": "success",
  "message": "Status pengiriman kurir Budi Santoso sukses diperbarui."
}
